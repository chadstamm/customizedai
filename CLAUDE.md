# CustomizedAI

Custom AI instructions generator. Users upload foundation documents (writing codex, personal constitution, story bank) and receive tailored AI system instructions they can use with any model.

## Tech Stack
- Next.js / React
- Claude API (Anthropic) — Opus for final generation
- Deployed on Vercel
